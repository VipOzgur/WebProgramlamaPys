# Personel Yönetim Web Uygulaması

Bu basit web uygulaması, bir organizasyondaki personel bilgilerini yönetmek için geliştirilmiştir. Uygulama, personel ekleme, güncelleme, silme ve listeleme gibi temel işlevleri sunar.

---

## Özellikler
- **Personel Listeleme:** Tüm personelin detaylı listesini görüntüleyin.
- **Personel Ekleme:** Yeni personel bilgilerini sisteme kaydedin.
- **Personel Güncelleme:** Mevcut personel bilgilerini düzenleyin.
- **Personel Silme:** İlgili personeli sistemden kaldırın.

- Teknolojiler
Backend: .NET 7.0
Frontend: Razor Pages
Veritabanı: SQLite/MsSql
Sürüm Kontrol: Git & GitHub
